# 💰 Sistema Financeiro

Projeto desenvolvido para praticar HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar movimentações financeiras
- Validar campos do formulário
- Atualizar saldo automaticamente
- Exibir histórico de movimentações

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar

1. Clone o repositório

```bash
git clone https://github.com/Elecktry/sistema-financeiro.git
```

2. Abra o arquivo `index.html`
