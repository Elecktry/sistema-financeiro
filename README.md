# 💰 Sistema Financeiro

Projeto desenvolvido para praticar **HTML, CSS e JavaScript**, simulando um sistema simples de controle financeiro com cadastro de movimentações, cálculo de saldo e persistência de dados no navegador.

## ✨ Funcionalidades

* Adicionar movimentações financeiras
* Definir o tipo da movimentação (**receita** ou **despesa**)
* Validar campos do formulário
* Atualizar saldo automaticamente
* Exibir histórico de movimentações
* Excluir movimentações da lista
* Salvar dados no **LocalStorage**
* Recarregar as movimentações mesmo após atualizar a página

## 🛠 Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **LocalStorage**

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Elecktry/sistema-financeiro.git
```

2. Acesse a pasta do projeto
3. Abra o arquivo `index.html` no navegador
   ou utilize a extensão **Live Server** no VS Code
