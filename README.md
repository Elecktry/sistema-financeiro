# 💰 Sistema Financeiro

[![Deploy na Vercel](https://img.shields.io/badge/Deploy-Vercel-black?logo=vercel)](https://sistema-financeiro-lac.vercel.app/)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5\&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3\&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript\&logoColor=black)

Aplicação web desenvolvida para controle de movimentações financeiras, permitindo cadastrar receitas e despesas, acompanhar o saldo e visualizar os gastos por categoria.

O projeto foi criado para praticar **HTML, CSS e JavaScript**, com foco em manipulação do DOM, regras de negócio, organização de código e persistência de dados no navegador.

---

## 🌐 Demonstração online

Acesse o projeto publicado na Vercel:

[🔗 Abrir Sistema Financeiro](https://sistema-financeiro-lac.vercel.app/)

---

## ✨ Funcionalidades

* Cadastro de movimentações financeiras
* Seleção entre receita e despesa
* Seleção de categoria
* Validação dos campos do formulário
* Cadastro por meio de modal
* Cálculo automático do saldo
* Exibição do total de receitas
* Exibição do total de despesas
* Atualização automática dos cards financeiros
* Histórico de movimentações
* Destaque visual para receitas e despesas
* Exclusão individual de movimentações
* Persistência de dados com LocalStorage
* Recuperação dos dados após atualizar ou fechar a página
* Gráfico de despesas por categoria

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Elecktry/sistema-financeiro.git
```

### 2. Entre na pasta do projeto

```bash
cd sistema-financeiro
```

### 3. Abra o projeto

Abra o arquivo `index.html` diretamente no navegador ou utilize a extensão **Live Server** no Visual Studio Code.
